| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-03-05-01 | Denial of Service / Tampering | Popup key leak: Enter inserts newline / Esc blurs / arrows pan while popup open | mitigate | Popup branch handled FIRST in keyPressed (before editor newline + before the 03-03 Esc-blur); all popup keys consumed (return true). UAT step 5 verifies. |
| T-03-05-02 | Denial of Service | Pathological tokenizer input (very long line, regex backtracking) | mitigate | TOKEN_PATTERN anchored `$` with a simple char class (no catastrophic backtracking); symbol set ≤ ~50 entries; O(n) scan; results capped at 6. |
| T-03-05-03 | Tampering / Injection | Suggestion text or signature rendered as a format string | accept | Drawn via `drawTextWithShadow` as literal text from a static symbol table; no user data formatted into the popup beyond the typed prefix (also literal). |
| T-03-05-04 | Tampering | Symbol-set drift: suggestions advertise a pathmind.* function that does not exist | mitigate | Names locked to PathmindBindings.java:55-63 (asserted in SuggestionEngineTest and javadoc); auto-derive preferred if a key-enumeration is exposed. |
| T-03-05-05 | Tampering | renderOverlay draws outside intended bounds (post-scissor) | mitigate | Popup anchored to the node body and clamped; overlay call is wrapped in NodeGraph's try/catch (03-02). Client-side cosmetic surface only. |
