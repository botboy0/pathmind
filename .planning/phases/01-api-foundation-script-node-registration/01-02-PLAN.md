## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| preset JSON file → NodeGraphPersistence.loadGraph | Preset files are user/third-party content; extraFields is an opaque blob written by an addon serializer |
| addon executor → game thread | Addon-supplied executor runs during graph execution; must not block or crash the tick thread |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-06 | Tampering | NodeGraphPersistence load (malicious/arbitrary extraFields blob) | accept | Pathmind never interprets extraFields itself — it only round-trips the map and hands it back to the addon's own deserialize. No code path treats blob contents as executable or as a Pathmind field (RESEARCH.md Security Domain) |
| T-01-07 | Denial of Service | Node.executeAddonNode (addon executor blocks or hangs) | mitigate (Phase 1 scope) | Phase 1 executor is graceful no-op (completedFuture); the dispatch returns the future without blocking the calling thread. | T-01-08 | Denial of Service | NodeGraph addon body renderer throws during per-frame render | mitigate | Wrap the `getBodyRenderer().render(...)` call in a try/catch that logs once and falls back to the grayed placeholder, so a buggy renderer cannot break the editor frame |
| T-01-09 | Tampering | Save writes a broken ADDON record (addonTypeId null) | mitigate | Save loop skips ADDON nodes with null addonTypeId and logs a warning (Pitfall 5) — no malformed record reaches the preset |
| T-01-SC | Tampering | npm/pip/cargo installs | accept | No package-manager installs in this plan (RESEARCH.md Package Legitimacy Audit — zero new external deps) |
